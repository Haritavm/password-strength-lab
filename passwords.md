# Password Strength Results

| Password              | Score/Strength | Feedback from Tool                  |
|-----------------------|----------------|-------------------------------------|
| password              | Very Weak      | Common word, easy to guess          |
| 123456                | Very Weak      | Short, only numbers                 |
| Password123           | Weak           | Predictable pattern                 |
| P@ssw0rd!             | Medium         | Substitution but still predictable  |
| Sunshine#45           | Medium         | Word + symbol + numbers             |
| My$ecureP@ssw0rd2025! | Strong         | Long phrase, mixed case, symbols    |
| Xy!9#QwT7@Lm$2zP      | Very Strong    | Random, complex, long               |
